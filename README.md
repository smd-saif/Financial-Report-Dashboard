# Financial Sales Dashboard

 ## Overview

The "Financial Sales Dashboard" is an interactive Power BI project designed to analyze sales and profit performance across different products, countries, customer segments, and discount bands. The dashboard helps users explore key business metrics through interactive visualizations and filters, enabling better business decision-making.
